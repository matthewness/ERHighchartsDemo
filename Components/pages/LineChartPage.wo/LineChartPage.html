Line Chart

Line Chart generated from the Model (er.highcharts.demo.model) Company Departments.
Line charts use values in the 
series.data
, so this chart maps the 
data
option in the series to the 
monthlyResults
key path.
Check and un-check the categories to redraw the chart with and without that line series.

Alternative bindings

Alternatively, you can specify the 
seriesNameKeyPath
and 
seriesDataKeyPath
bindings.
Here the value "name" is bound to the 
seriesNameKeyPath
binding, so each line will be defined by each name value of each department.
Likewise the value "monthlyResults" is bound to the 
seriesDataKeyPath
binding, so each line will have those numbers as its values.
Different approach, same result.

Build your own line

Build your own line chart by entering the details in the form and hitting build.
The field is the line y-value value as a number. The x axis is the date time you entered the number.
