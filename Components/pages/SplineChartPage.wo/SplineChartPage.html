Spline Curve Chart

Spline Curve Chart generated from the Model (er.highcharts.demo.model) Company Departments.
Spline Curve charts use values in the 
series.data
, so this chart maps the 
data
option in the series to the 
monthlyResults
key path.

Alternative bindings

Alternatively, you can specify the 
seriesNameKeyPath
and 
seriesDataKeyPath
bindings.
Different approach, same result. See Line Chart Page for details.

Build your own spline curve chart

Build your own spline curve chart by entering the details in the form and hitting build.
The field is the line y-value value as a number. The x axis is the date time you entered the number.
