Column Chart

Column Chart generated from the Model (er.highcharts.demo.model) Company Departments.
Bar charts use values in the 
series.data
, so this chart maps the 
data
option in the series to the 
monthlyResults
key path.
We also set the categories binding which maps to the 
xAxis.categories
Highcharts option. These are the month labels on the x-axis.

Alternative bindings

Alternatively, you can specify the 
seriesNameKeyPath
and 
seriesDataKeyPath
bindings.
Different approach, same result. See Line Chart Page for details.
