Scatter Chart

Scatter Chart generated from the Model (er.highcharts.demo.model) Company Fleet Vehicles Fuel Economy.
Scatter charts use values in the 
series.data
, so this chart maps the 
data
option in the series to the 
values
key path of the FleetVehicleFuelEconomy objects.
Because the data for this chart is created randomly at runtime, this example doesn't quite show the power of the Scatter Chart. You'll have to imagine!
Note, also, that you can create a marquee in the chart to zoom in to your selected area. Click on Reset zoom to get the original chart.

Alternative bindings

Alternatively, you can specify the 
seriesNameKeyPath
and 
seriesDataKeyPath
bindings.
