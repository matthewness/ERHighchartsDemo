Bar Chart

Bar Chart generated from the Model (er.highcharts.demo.model) Company Department financial details.
Bar charts use values in the 
series.data
, so this chart maps the 
data
option in the series to the 
values
key path of the DepartmentBreakdown objects.
We also set the categories binding which maps to the 
xAxis.categories
Highcharts option. These are the department names.
Note here the floating legend.

Alternative bindings

Alternatively, you can specify the 
seriesNameKeyPath
and 
seriesDataKeyPath
bindings.
Different approach, same result. See Line Chart Page for details.
